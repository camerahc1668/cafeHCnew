🛒 Quy trình bán hàng tại bàn

1
Vào Tab Bàn → nhấn vào bàn muốn gọi món (xanh = trống, đỏ = đang có khách)
2
Panel giỏ hàng hiện ra bên phải → chọn món từ danh sách hoặc dùng nút giỏ hàng 🛒 trên header
3
Nhấn 🍳 Gửi bếp để bếp nhận đơn, hoặc ✅ Thanh toán ngay
4
Chọn phương thức: Tiền mặt / Chuyển khoản / Công nợ → xác nhận
🛵 Đơn mang về / App giao hàng
Nhấn + Đơn mới ở mục "Đơn mang về" → chọn kênh (GrabFood, Shopee...) → chọn món → thanh toán
📱 QR Order (khách tự gọi)
Nhấn icon QR trên thẻ bàn → khách quét QR → chọn món trên điện thoại → đơn tự động vào bếp

📦 Quản lý kho

▸
Thêm nguyên liệu: tên, đơn vị cơ sở, tồn kho, định mức cảnh báo

▸
Tự động trừ kho: khi thanh toán đơn → kho nguyên liệu tự trừ theo công thức món
▸
Cảnh báo: nguyên liệu sắp hết hiện màu đỏ/vàng

📋 Cách lấy Firebase Config — 4 bước
1
Tạo project Firebase
Vào trang console, tạo project mới (hoặc dùng project cũ)
Mở Firebase Console ↗
2
Bật Realtime Database
Build → Realtime Database → Create database → Start in test mode
Sau đó vào Rules, dán đoạn sau rồi nhấn Publish: 
{
  "rules": {
    ".read": true,
    ".write": true
  }
}
3
Lấy Config từ Project Settings
① Nhấn Project Settings (bánh răng trên menu trái)
② Kéo xuống mục Your apps → chọn Web app </>
③ Nếu chưa có → nhấn Add app → Web → đặt tên bất kỳ
④ Copy đoạn firebaseConfig = { ... }
⑤ Chỉ lấy phần JSON bên trong { } dán vào ô trên
4
Điền Shop ID & Lưu cài đặt
Điền Shop ID ở trên → nhấn Lưu cài đặt → nhấn In QR trên bàn → khách quét là nhận đơn ngay!
✅ Miễn phí hoàn toàn với Spark Plan (0 USD/tháng)
📱 Luồng khách quét QR order
1
Thu ngân nhấn icon QR trên thẻ bàn → QR code hiện ra
2
Khách quét QR bằng camera điện thoại → mở trang menu của quán
3
Khách chọn món → nhấn "Gửi đơn lên bếp" → đơn ghi vào Firebase
4
POS nhận đơn realtime → hiện chuông thông báo → vào tab Bếp để xác nhận
✅ Điều kiện để hoạt động:
⬜ Link web công khai đã điền
⬜ Firebase Shop ID đã điền
⬜ Firebase Config JSON hợp lệ
